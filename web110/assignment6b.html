Activity 1 What is accessibility? Learning about accessibility was interesting as it mentioned a disability I have which is color blindless. Websites that color code things in colors I can’t see I tend to try to find an alternative for as I usually can’t tell the difference between the colors. I hadn’t thought about many other visual impairments, but I can see how things like Jaws screen reader can be useful, however specifically the Jaws screen reader kind of hurt my ears and I really hope there are better sounding screen readers out there. This lesson was a good reminder that if I want to make websites that anyone, and everyone can use, I will have to put myself in other people’s shoes, making sure to add options for a broad assortment of possible disabilities.
